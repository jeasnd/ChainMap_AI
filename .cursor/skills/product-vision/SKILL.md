# 产品愿景 · AI 专注投资站

## 一句话

**专注 AI 投资**：实时（或定期更新）呈现 AI 产业链上**当前最值得关注的宏观方向**，在每个方向下给出 **美股 / A 股 / 港股** 中最值得研究的 **1～3 只**标的，并用**可核对的数据与逻辑**说明为何选它。

## 三层能力

| 层级 | 用户得到什么 | 站点对应 |
|------|----------------|----------|
| 1. 宏观热点 | 当前 AI 资金/叙事最热的环节，新热在上、旧热在下，带时间戳 | `hot-topics.html` + `data/hot-topics.json` |
| 2. 精选标的 | 选定方向后，各市场 1～3 只「壁垒高、盈利能力强、难替代」的公司 | `topic.html?id=` + `data/stocks.json` |
| 3. 数据论证 | 点击个股查看指标、产业链位置、选股理由与风险 | `stock.html?id=` |

## 用户路径

```
首页 → 热门方向（按更新时间排序）→ 点击某方向 → 看该方向 Top 标的
     → 点击个股 → 看论证页（逻辑 + 指标 + 风险）
```

## 选股价值观（全站一致）

优先顺序（用于 Agent 更新数据时）：

1. **产业链位置**：是否处在资本扩散路径上的「当前主战场」（参考六层框架，见 `macro-hot-topics`）
2. **壁垒**：技术、份额、客户锁定、规模效应
3. **盈利与现金流**：毛利率、ROE、经营现金流（能拿到则用财报/一致预期）
4. **难替代性**：切换成本、标准生态、专利与产能
5. **跨市场覆盖**：同一逻辑尽量给出 US / A / HK 各 0～1 只代表，避免只堆美股

## 合规与表述

- 全站为**研究框架与教育**，不是证券投顾；页面必须有免责声明。
- 禁止「必涨」「稳赚」；用「市场常讨论」「当前叙事热度」「框架内优选」等表述。
- 数据需标注**来源与 as-of 日期**；抓取失败时显示「待更新」而非编造。
