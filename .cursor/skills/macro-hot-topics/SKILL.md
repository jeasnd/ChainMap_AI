# 宏观热点 · AI 方向发现与排序

## 六层产业链框架（资本扩散路径）

自上而下为资金从「算力核心」向外扩散的常见叙事（与用户参考图一致）：

| 层 | 名称 | 典型主题 | 扩散阶段关键词 |
|----|------|----------|----------------|
| L1 | 算力核心层 | GPU / 加速芯片 | 已爆发、估值已反映部分预期 |
| L2 | 存储层 | HBM / DRAM | 主升浪、供需紧、弹性大 |
| L3 | 互连通信层 | 光模块 / 交换机 / 网络 | 加速扩张、量价齐升 |
| L4 | 基础设施层 | 数据中心 / 电力设备 / 液冷 | 稳步扩散、REITs 与设备 |
| L5 | 能源与公用事业层 | 核电 / 燃气 / 电网 | 长线布局、用电需求 |
| L6 | AI 应用与软件层 | Agent / 企业软件 / 垂直应用 | 价值爆发、长期 Alpha |

## 什么是「值得投资的话题」（宏观）

满足 **≥2 条** 可写入 `hot-topics.json`：

- 近 7～30 日资本开支、订单、涨价、政策或龙头财报**验证**了景气
- 主流研报 / 财经媒体**高频**讨论且逻辑可复述（非单日谣言）
- 相对上一层存在**扩散关系**（例如算力 capex → 存储 → 光模块）
- 有**可跟踪指标**（HBM 报价、GPU 出货、DC 上架功率、电价等）

## 热度评分 `heatScore`（0～100）

| 因子 | 权重建议 | 说明 |
|------|----------|------|
| 叙事新鲜度 | 25% | 首次进入 Top 列表时间越近越高 |
| 数据验证 | 35% | 财报、指引、行业数据是否印证 |
| 资金/估值弹性 | 20% | 相对已定价程度（L1 低、L2～L3 常高） |
| 持续天数 | 20% | 热点是否连续多周在榜 |

## 列表排序规则（前端已实现）

1. **主排序**：`updatedAt` 降序（最近更新的在最上）
2. **次排序**：`heatScore` 降序
3. **归档**：`archived: true` 的条目沉底，仍保留时间戳供回顾

## Agent 更新 checklist

- [ ] 新热点是否分配唯一 `id`
- [ ] `updatedAt` 是否改为当前时间
- [ ] 旧热点仅降 `heatScore` 或设 `archived`，不删除历史
- [ ] `stockIds` 是否与 `stocks.json` 一致
